# budgeting-app

A bucket/envelope based budgeting app, similar to Up or Actual.

## Tech Stack
- Next.js
- Shadcn/ui
- Bun

## Features
- A total balance number.
- "Buckets" - you can create them, you can delete them, they have a monetary balance. The homepage will show a list of the buckets.
- You can set how much money each bucket has through an input box.
- Add payment button - you add transactions that take money out of a bucket, and out of a total balance.
- Add deposit button - add transactions that put money into total balance.
- Unallocated money is displayed at the top of the screen. Unallocated money is balance - total of all buckets.
- Each transaction is stored onto the server.
